# Bike Sales Dashboard (Microsoft Excel)
## Project Overview

This project analyzes a bike sales dataset using Microsoft Excel to understand customer purchasing behavior. The dashboard provides insights into the factors that influence bike purchases and presents key findings through interactive visualizations.

## Dataset

The dataset includes customer demographic and lifestyle information, such as:

- Age
- Gender
- Marital Status
- Education
- Occupation
- Income
- Commute Distance
- Region
- Purchased Bike (Yes/No)

 ## Tools Used
- Microsoft Excel
- Excel Tables
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design
  
 ## Data Preparation

- Checked data quality
- Formatted data for analysis
- Organized the dataset into an Excel Table
- Prepared the data for Pivot Table analysis

## Analysis Performed

- Average income comparison by gender
- Customer commute distance based on count of bike purchases
- Customer bike purchases by age brackets

 ## Dashboard

The interactive dashboard includes:

- Pivot Charts
- Slicers for filtering data
- Customer purchase analysis
- Visual comparison of purchasing trends
## Key Skills Demonstrated
- Data Cleaning
- Data Analysis
- Pivot Table Analysis
- Dashboard Development
- Data Visualization
- Business Reporting
